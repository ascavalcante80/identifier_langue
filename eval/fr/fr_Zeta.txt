Il est possible que l'alphabet arménien dérive de l'alphabet grec. Dans ce cas, le za Զ dériverait du zêta.


== Codage ==
La majuscule Ζ possède les codages suivants :
Unicode : U+0396
Entité HTML : &Zeta;
TeX : \Zeta ; 
DOS Greek : 133
DOS Greek-2 : 169
Windows-1253 : 198
La minuscule ζ possède les codages suivants :
Unicode : U+03B6
Entité HTML : &zeta;
TeX : \zeta ; 
DOS Greek : 157
DOS Greek-2 : 218
Windows-1253 : 230

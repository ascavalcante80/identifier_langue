Perincian server menurut lokasinya adalah sebagai berikut (sebelum Januari 2013).
Tampa, Florida:
14 buah server berbasisdata MySQL,
18 buah server Squid,
93 buah server Apache, dan
12 buah server lainnya untuk beragam keperluan.

Paris, Perancis:
3 buah server Squid.

Amsterdam, Belanda:
12 buah server.

Yahoo!:
23 buah server.

Estas herramientas son aplicaciones web creadas con lenguajes de programación como PHP y que usan bases de datos como MySQL que no solamente permiten una edición más cómoda de los contenidos, sino también implementar webs dinámicas.
Entre estos sistemas destacan las herramientas para la creación de blogs por ser las más populares, lideradas por el CMS WordPress.
WordPress en particular, además de ser la herramienta líder para la creación de blogs, se está conviertiendo en la herramienta más utilizada para la creación de webs pequeñas y medianas.


== Posicionamiento web (SEO) ==
El posicionamiento web de una página web es la base para optimizar todo un sitio web el cual es un conjunto de páginas web.
La finalidad es que la página sea lo más inteligible posible para los bots de los buscadores, lo cual ayudará a que se posicionen más alto en las búsquedas para las cuales están optimizadas.

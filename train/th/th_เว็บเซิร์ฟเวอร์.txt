เว็บเซิร์ฟเวอร์ สามารถมีได้ 2 ความหมาย คือ
เครื่องคอมพิวเตอร์ซึ่งให้บริการเว็บไซต์ ผู้ใช้เรียกชมหน้าเว็บไซต์ได้โดยใช้โพรโทคอล HTTP ผ่านทางเว็บเบราว์เซอร์
โปรแกรมคอมพิวเตอร์ ที่ทำงานตามคุณสมบัติข้างต้น


== การใช้งาน ==
เมื่อผู้ใช้ป้อนยูอาร์แอล (URL) ในโปรแกรมเว็บเบราว์เซอร์
เครื่องไคลแอนท์จะแปลงชื่อโฮสต์ ภายในยูอาร์แอลเป็นไอพีแอดเดรส
เครื่องไคลแอนท์ติดต่อกับเครื่องเว็บเซิร์ฟเวอร์ โดยปรกติจะใช้โพรโทคอล TCP พอร์ต 80
เมื่อทำการเชื่อมต่อเสร็จ จะใช้โพรโทคอล HTTP ในการเรียกใช้ข้อมูลที่ต้องการ


== ซอฟต์แวร์ ==
ซอฟต์แวร์เว็บเซิร์ฟเวอร์ที่ได้รับความนิยมสูงสุด 4 อันดับแรก คือ
Apache HTTP Server จาก Apache Software Foundation
Internet Information Server (IIS) จากไมโครซอฟท์
Sun Java System Web Server จากซัน ไมโครซิสเต็มส์ (เดิมช
